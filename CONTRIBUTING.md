# Contributing to PAM Toolkit

Thanks for your interest! All contributions are welcome.

## How to Contribute
1. Fork the repo
2. Create a branch ('git checkout -b feature/xyz')
3. Commit your changes
4. Open a Pull Request

## Code Style
- Python: PEP 8
- YAML: 2-space indent
- All scripts must have demo mode

We follow the standard GitHub Flow.
